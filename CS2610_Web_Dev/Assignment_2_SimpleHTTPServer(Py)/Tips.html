Bash Tips

Bash commands

Some useful Bash commands:

  • ls - List Directory
  • cd - change directory
  • mkdir - make a directory
  • touch - create a fie
  • echo - print text to the terminal window
  • pwd - print working directory

An example using the ls directory:

    $ cd ..

This will take you to the parent directory.

Git Tips

Git commands

Some useful Git commands:

  • -git commit -a -m "info about commit" ~tell git to commit all files in directory with message in quotes
  • -git status ~ask git about the status of files in the directory
  • -git add . ~have git add all files in the directory
  • -git add {file} ~have git add a file

An example of using the git status and what it might return:

    $ git status

    On branch master
    Your branch is ahead of 'origin/master' by 1 commit.
    (use "git push" to publish your local commits)

    Changes not staged for commit:
    (use "git add <file>..." to update what will be committed)
    (use "git restore <file>..." to discard changes in working directory)
        modified:   .idea/workspace.xml
        modified:   Tips.html

        Untracked files:
        (use "git add <file>..." to include in what will be committed)
            Jack Sailing.jpg
            Jack Shooting.jpg
            Jack Drinking.html

            no changes added to commit (use "git add" and/or "git commit -a")

    $

HTML Tips

HTML tags

Some useful HTML tags:

  • title -The title of the page
  • h1 -The largest default heading in HTML
  • h2 -The second largest heading in HTML
  • p -A standard paragraph
  • code -Defines a piece of computer code
  • xmp -Allows exact sharing of code without running the HTML scripts

Lists

Ordered List

An ordered list will look like:

  1. First list element
  2. Second list element
  3. And so on..

The code to produce this looks like:

    <ol>
        <li>First list element</li>
        <li>Second list element</li>
        <li>And so on..</li>
    </ol>

Unordered List

An unordered list will look like:

  • First list element
  • Second list element
  • And so on..

And the code to produce this looks like:

    <ul>
        <li>First list element</li>
        <li>Second list element</li>
        <li>And so on..</li>
    </ul>

Hyperlinks

To hyperlink to a picture, use the "a" tag. The code to insert this picture looks like:

    <a href="Jack%20Sparrow%20-%20lick.jpg">Jack Sparrow</a>

CSS Tips

CSS concepts

Remember:

  • No HTML tags
  • All classes begin with a . ex, .noPadding
  • An ID tag is referenced via #

An example of a css class:

    img {
        max-width: 80%;
        display: block;
        height: auto;
        margin-left: auto;
        margin-right: auto;
    }
